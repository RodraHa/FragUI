# Color

La paleta de color prioriza contraste, claridad semántica y funcionalidad. Se compone de una base neutral dominante en escala de grises para estructura, fondo, texto y bordes, y de una serie de paletas de acento reservadas para comunicar estado, intención o interacción.

Si un elemento no comunica una acción o un estado apropiado, no debe usar color de acento.

## Regla de uso

El color es el tercer nivel de la jerarquía visual. Primero se debe resolver la estructura con espacios y tipografía; luego, si el significado lo exige, se usa acento cromático.
