# Espaciado

El sistema de espaciado utiliza una progresión geométrica: la diferencia entre los valores iniciales es pequeña, pero aumenta proporcionalmente en los niveles sucesivos de la escala. Como resultado, los cambios de separación son perceptibles para el usuario y permiten construir jerarquía visual sin depender de valores arbitrarios.

En FragUI, el espaciado es el primer recurso para organizar información. Antes de intensificar color o peso tipográfico, se debe resolver la jerarquía mediante distribución, respiración y proximidad entre elementos.

## Criterios de uso

- Priorizar relaciones consistentes entre contenedor, contenido y acciones.
- Usar espaciado pequeño para elementos íntimamente relacionados.
- Usar saltos amplios para separar grupos semánticos distintos.
- Evitar márgenes y paddings improvisados por caso específico.

## Regla de jerarquía

La jerarquía visual debe construirse en orden descendente usando espacios, tipografía y color.
