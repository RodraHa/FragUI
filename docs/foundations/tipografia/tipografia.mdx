# Tipografía

La elección tipográfica se fundamenta en legibilidad, consistencia y capacidad jerárquica. En FragUI se utiliza Satoshi, una fuente de carácter geométrico y neutral, adecuada para interfaces digitales por su alta legibilidad y ausencia de serifas.

Al mismo tiempo, se define una escala tipográfica contenida que permite distinguir niveles de información sin que el aumento resulte disruptivo para la lectura. La diferencia entre tamaños debe ser perceptible, pero nunca estridente.

También se limita la cantidad de pesos tipográficos para evitar ruido visual. El peso debe reforzar la jerarquía, no reemplazarla.

## Familia tipográfica

- `font-family-base`: {typography.type.primary}

## Criterios de uso

- Usar tamaño para construir jerarquía antes de recurrir a color.
- Reservar pesos altos para énfasis puntual o encabezados.
- Evitar mezclar demasiados pesos en un mismo componente.
- Mantener una lectura cómoda en estados, etiquetas, botones y mensajes.
